## 1. Список доработок
- Интеграционные тесты для проверки end‑to‑end сценариев (UC1–UC9).
- Юнит‑тесты для ключевых функций (распознавание ингредиентов, генерация рецептов, работа с избранным).
- UX‑улучшения: удобный просмотр истории, отзывов и избранного, оптимизация форм регистрации/входа.

---

## 2. Приоритизация задач

| Задача | Приоритет | Комментарий |
|--------|-----------|-------------|
| Интеграционные тесты | P1 | Критично для подтверждения DoD и защиты |
| Юнит‑тесты | P1 | Обеспечивают надёжность и покрытие функций |
| UX‑улучшения | P2 | Повышают удовлетворённость пользователей |

---

## 3. Распределение по ролям

| Роль | Ответственность |
|------|-----------------|
| **System Architect (SA)** | Проверить достижение целей из Лабы 1, проанализировать результаты тестирования, скорректировать требования и подготовить план доработок для финального спринта |
| **Fullstack Developer** | Провести комплексное UX‑тестирование интерфейса, тестирование FastAPI endpoints, исправить все найденные баги и оптимизировать пользовательский опыт |
| **MLops Engineer** | Финализировать CI/CD pipeline с полными качественными гейтами, настроить Grafana‑дашборды для мониторинга production, провести нагрузочное тестирование и оптимизировать производительность |
| **AI Engineer** | Провести A/B‑тесты различных промптов и моделей, собрать метрики качества AI (accuracy, hallucination rate), оптимизировать промпты на основе результатов и подготовить итоговый отчёт |
